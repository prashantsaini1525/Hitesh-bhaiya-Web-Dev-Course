# Section 3 – Basics of Web Development

## Lecture: Frontend backend and APIs

## 1. Frontend vs Backend Overview

- **Frontend:** Everything user sees and interacts with.

  - Core technologies: **HTML, CSS, JavaScript**
  - Additional tools: **CSS frameworks** like Bootstrap, Tailwind
  - JS frameworks/libraries: React, Vue, Svelte, etc.
  - Learning HTML + CSS + JS = Junior Frontend Engineer level

- **Backend:** Server-side logic and data management.
  - Programming languages: Node.js (JavaScript), PHP, Python (Django), Ruby on Rails, etc.
  - Database: Stores information (MySQL, MongoDB, PostgreSQL, etc.)
  - Learning a backend language + database = Junior Backend Engineer level

## 2. APIs (Application Programming Interface)

- Acts as a **bridge** between frontend and backend.
- Allows communication between systems written in different languages.
- Data exchange format usually **JSON**.
- Analogy: Frontend = customer, Backend = kitchen, API = waiter, JSON = plate of food

## 3. Flow of Data

1. User interacts with **frontend** (browser).
2. Frontend sends request to **API endpoint**.
3. Backend receives request, processes it, communicates with **database** if needed.
4. Backend sends response back through **API** in **JSON format**.
5. Frontend receives response and updates UI accordingly.

## 4. Key Takeaways

- Frontend = presentation layer (HTML, CSS, JS + frameworks/libraries)
- Backend = logic + data layer (Programming language + Database)
- API = communication layer between frontend & backend
- Databases can be **SQL (MySQL, PostgreSQL)** or **NoSQL (MongoDB)**
- Full web app = Frontend + Backend + API + Database

---

## Hinglish Quick Notes

- **Frontend:** Jo user dekhta/ use karta hai → HTML, CSS, JS
- **Backend:** Server side ka logic + database → Node.js, Python, PHP, MySQL, MongoDB
- **API:** Frontend aur Backend ka bridge, data exchange JSON mein
- **Flow:** Browser → API → Server → DB → Server → API → Browser
- SQL = table format, NoSQL = JSON format
- Frontend + Backend + API + DB = Web application ka full picture
