Lists

What is lists?

Lists are used to group related items together in a structured way. They can be ordered or unordered.

Unordered List

An unordered list is a collection of items that do not have a specific order. It is typically represented with bullet points.

  • Item 1
  • Item 2
  • Item 3

Ordered List

An ordered list is a collection of items that have a specific order. It is typically represented with numbers or letters.

  1. First item
  2. Second item
  3. Third item

Definition List

A definition list is used to define terms and their corresponding definitions. It consists of <dl> (definition list), <dt> (definition term), and <dd> (definition description) elements.
