# HTML Quiz with Answers

## Question 1
**What does HTML stand for?**
✅ **Answer:** Hypertext Markup Language
**Explanation:** HTML (Hypertext Markup Language) is the standard language used to create and structure web pages. It describes the content and layout using tags.

---

## Question 2
**Which of the following is a block-level element in HTML?**
✅ **Answer:** `<div>`
**Explanation:** Block-level elements take up the full width available and start on a new line, such as `<div>`, `<p>`, and `<section>`.

---

## Question 3
**What is the main purpose of Emmet in web development?**
✅ **Answer:** To create shortcuts for writing HTML and CSS code
**Explanation:** Emmet allows developers to write shorthand syntax that expands into full HTML or CSS code, boosting productivity and speed.

---

## Question 4
**Which of the following tags is used to create a table row in HTML?**
✅ **Answer:** `<tr>`
**Explanation:** The `<tr>` tag defines a row in an HTML table. Inside `<tr>`, you use `<th>` for headers and `<td>` for data cells.

---

## Question 5
**What does the action attribute in an HTML form do?**
✅ **Answer:** Specifies where to send the form data when the form is submitted
**Explanation:** The `action` attribute defines the URL or endpoint where form data will be sent when submitted.

---

## Question 6
**What is a semantic HTML element?**
✅ **Answer:** An element that clearly describes its meaning to both the browser and the developer
**Explanation:** Semantic elements like `<article>`, `<header>`, and `<footer>` provide meaning about their content, improving readability and accessibility.

---

## Question 7
**Which of the following is an example of a semantic HTML element?**
✅ **Answer:** `<section>`
**Explanation:** `<section>` represents a distinct part of a document, making content more structured and understandable.

---

## Question 8
**Which tag is used to include audio in an HTML document?**
✅ **Answer:** `<audio>`
**Explanation:** The `<audio>` tag embeds sound content in a webpage. It supports attributes like `controls`, `autoplay`, and `loop`.

---

## Question 9
**What does ARIA stand for in the context of web accessibility?**
✅ **Answer:** Accessible Rich Internet Applications
**Explanation:** ARIA provides attributes that make web applications more accessible to people using assistive technologies.

---

## Question 10
**Which ARIA attribute is used to define a landmark region in a web page?**
✅ **Answer:** `role`
**Explanation:** The `role` attribute defines the purpose or type of an element (e.g., `role="navigation"`, `role="main"`), helping assistive technologies navigate web content.
